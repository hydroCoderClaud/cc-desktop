企业微信桥接配置指南

将企业微信智能机器人接入 Hydro Desktop，支持原生流式回复和桌面主动推送

钉钉 飞书 企业微信 微信
📋 前提条件
  • Hydro Desktop 已安装并正常运行
  • 拥有企业微信管理后台的管理员权限
⚠️ 重要限制 一个智能机器人只能绑定一个桌面端。多台桌面端同时连接同一个机器人会导致消息丢失或重复回复。
1 创建企业微信智能机器人
  1. 登录 企业微信管理后台
  2. 进入「应用管理」→「智能机器人」→「创建智能机器人」
  3. 填写机器人名称和描述
  4. 创建完成后，在机器人详情页获取 Bot ID 和 Secret
⚠️ 注意 这里的 Bot ID 和 Secret 仅用于消息收发桥接，不要用于后文的 wecom-cli 联系人读取。
2 配置机器人权限
权限用途
消息收发基础权限，创建机器人即默认开通
媒体文件上传转发图片到企业微信
3 在 Hydro Desktop 中配置
  1. 打开 Hydro Desktop，点击左下角齿轮图标，选择「企业微信设置」
  2. 填写基本配置：
字段说明
启用企业微信桥接打开开关
Bot ID填入第一步获取的 Bot ID
Secret填入第一步获取的 Secret
默认工作目录Agent 执行命令的工作目录（可选，强烈建议设置）
历史会话数量选择历史会话时显示的最大数量（默认 5，范围 1-20）
  1. 点击「保存」，再点击「连接」
  2. 确认状态变为 已连接
4 配置联系人获取（wecom-cli）
⚠️ 安全提醒 联系人读取请使用新建的独立机器人完成初始化，不要复用第三步中桥接机器人的 Bot ID/Secret。
步骤操作
1. 检测 CLI点击「检测 CLI」查看安装状态
2. 安装 CLI如未安装，点击「安装 CLI」打开终端执行安装
3. 初始化点击「打开初始化终端」，输入独立机器人的凭据完成接入
4. 授权如未弹出授权确认，去管理后台手动开通通讯录权限
5. 测试点击「测试读取联系人」确认权限生效
如果不需要桌面端主动给企业微信成员发消息，可跳过这一步。
5 验证配置
  1. 在企业微信中找到你的机器人，发送一条文本消息
  2. 观察 Hydro Desktop 是否出现企业微信会话
  3. 注意回复是逐段流式返回的（不同于钉钉/飞书的整段返回）
⚡ 双机器人说明
桥接机器人联系人读取机器人
用途收发消息读取通讯录
凭据填入基本配置 (Bot ID/Secret)wecom-cli init (终端)
权限要求消息收发、媒体上传通讯录读取
⌨️ 命令参考
命令说明
/help显示所有可用命令
/status查看当前会话状态与历史会话列表
/sessions列出当前聊天下的活跃会话
/new [目录]新建会话
/resume [编号]恢复历史会话
/rename <名称>重命名当前会话
/close [编号]关闭当前或指定会话
❓ 常见问题
连接失败确认 Bot ID/Secret 正确、智能机器人已创建并启用。
桌面端主动发送列表为空确认已完成 wecom-cli 安装/初始化、独立机器人的通讯录权限已授权。
图片无法转发确认已开通「媒体文件上传」权限、图片大小不超过 20MB。
收到重复回复检查是否有多台桌面端配置了同一个智能机器人。
