# Naive UI + Claude 主题测试

## 📖 说明

这是一个独立的测试页面，用于展示 Naive UI 组件库配合 Claude 官方风格主题的实际效果。

## 🚀 如何查看

### 方法 1：直接打开（推荐）

双击 `demo.html` 文件，用浏览器打开即可查看。

### 方法 2：本地服务器

```bash
# 在当前目录下
cd test-naive-ui

# 使用 Python 启动简单服务器
python -m http.server 8080

# 或使用 Node.js
npx serve .

# 然后在浏览器访问
http://localhost:8080/demo.html
```

## 📝 评估标准

### 如果视觉效果达到 85-90% 相似度：
- ✅ **可以考虑使用 Naive UI**
- 优势：30+ 组件开箱即用
- 劣势：+200KB 包体积

### 如果感觉差异明显：
- ✅ **建议自己写组件**
- 优势：100% 匹配，0KB 增加
- 劣势：需要自己实现（但只需 1-2 天）

### 混合方案：
- ✅ **基础组件自己写 + 复杂组件用 Naive UI**
- 最灵活的方案
- 基础 UI 100% Claude 风格
- 高级功能快速开发

## 🔍 需要关注的细节

1. **按钮的悬停效果**
   - 原生：平滑过渡 + 轻微阴影
   - Naive UI：是否相似？

2. **输入框的焦点状态**
   - 原生：橙色边框 + 浅橙色外发光
   - Naive UI：是否相似？

3. **卡片的阴影**
   - 原生：轻微阴影 `0 2px 8px rgba(0, 0, 0, 0.08)`
   - Naive UI：是否相似？

4. **整体和谐度**
   - 多个组件放在一起
   - 是否看起来统一？
